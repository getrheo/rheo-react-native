# rheo-react-native

React Native SDK for Rheo — [`@getrheo/react-native-expo`](https://www.npmjs.com/package/@getrheo/react-native-expo) and [`@getrheo/react-native-bare`](https://www.npmjs.com/package/@getrheo/react-native-bare).

**Compatibility:** `rheo-react-native@2.x` requires `@getrheo/contracts@2.x` and matching `@getrheo/flow-runtime@2.x` (install flavors pull the graph transitively).

**Examples:** [rheo-example-expo](https://github.com/getrheo/rheo-example-expo) · [rheo-example-bare](https://github.com/getrheo/rheo-example-bare)

```bash
pnpm install && pnpm verify
```
